## stocktest
获取股票主力资金流入数据

数据来源为：东方财富网主力资金流入数据

输出：
当日主力资金流入，3日主力资金流入和5日主力资金流入的交集；
没有使用10日资金是因为过于长期，对于主力资金动向反应过于滞后；

## hk_warning.py
